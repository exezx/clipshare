# Escape

A free clip sharing website.

Upload videos, get a link, and share instantly.

# Features

* Upload clips (max 100MB)
* Instant shareable links
* Discord embeds
* Editable clip names

# How It Works

1. Upload your clip
2. Get a link
3. Share it anywhere

# Tech

* Python (Flask)
* Hosted on Render

# Notes

* Files may be temporary (Render free tier)
* No login required
